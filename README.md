### 支出記帳網站
這是一個使用 Node.js + Express 打造的支出紀錄網站

### Features
1. 使用者可以在首頁看到所有支出紀錄
2. 使用者可以編輯支出紀錄
3. 使用者可以刪除任一支出紀錄
4. 使用者可以新增支出紀錄
5. 使用者可以透過下拉選單來篩選出特定類別的支出紀錄，上方金額也會顯示該類別的總金額
6. 使用者可以在首頁看到所有的支出金額

### Environment SetUp
Node.js

### Installation and Execution
1. 開啟 terminal，並Clone 此專案至本機電腦
2. 進入存放此專案的資料夾
3. 安裝套件 (npm install) 
4. 執行專案，並可使用瀏覽器輸入網址"http://localhost:3000"來開啟網頁
